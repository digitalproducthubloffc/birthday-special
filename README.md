# Aesthetic Birthday Celebration App 🎉✨

A beautiful, highly interactive, and fully responsive digital birthday celebration experience built with modern web technologies. This project was designed to be a deeply personal and visually stunning scrapbook-style gift, complete with ambient animations, custom floating elements, and a meticulously crafted mobile experience.

## ✨ Key Features

- **Aesthetic Scrapbook Layout**: A stunning hero section designed like a physical scrapbook, featuring overlapping polaroids, washi tape, and watercolor background blobs.
- **Immersive Animations**: Powered by Framer Motion, the app includes floating balloons, flapping butterflies, twinkling gold stars, falling flower petals, and a fun popping balloon effect.
- **Tailored Mobile Experience**: The mobile view (under 450px) completely reimagines the layout, introducing an exclusive **Glowing Circular Image Slider** and perfectly proportioned delicate icons (🌸 🧸 🎂 💗).
- **Flawless Responsiveness**: Custom-built media queries ensure the app looks gorgeous on everything from a tiny smartphone to a massive ultra-tall iPad Pro, preventing awkward gaps or overlapping text.
- **Interactive Mini-Player**: A beautifully styled, mock "music player" widget that sits elegantly at the bottom of the screen.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router & Turbopack)
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) & Vanilla CSS for complex media queries
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Components**: [Radix UI](https://www.radix-ui.com/)
- **Masonry Grid**: `react-masonry-css` for the memories gallery

   \`\`\`bash

   \`\`\`

3. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser to experience the magic.

## 🎨 Design Philosophy

This project prioritizes **Visual Excellence** above all else. Instead of generic flat colors, it heavily utilizes:
- Soft blush, rose, and gold linear/radial gradients.
- Glassmorphism effects and deep, realistic drop-shadows.
- Elegant typography pairings (Cormorant, Dancing Script, and Great Vibes).
- Micro-interactions and continuous ambient movement to make the webpage feel "alive."
